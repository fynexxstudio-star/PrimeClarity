This project is a minimalist, professional service website built with Next.js (App Router) and Tailwind CSS. It is designed for clarity, accessibility, and ease of deployment.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run dev server:
   ```bash
   npm run dev
   ```

## Features
- Next.js 14 with App Router
- Tailwind CSS for utility-first styling
- Responsive, accessible components
- Contact API route with basic validation
- SEO-friendly metadata

## Deployment
Deploy to Vercel or any platform that supports Next.js App Router. For a production build:

```bash
npm run build
npm run start
```
